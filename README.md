# Modern Planner

این مخزن شامل یک برنامه پلنر مدرن با رابط گرافیکی (Tkinter) است که می‌تواند کارها و رویدادها را مدیریت کند و نمای کلی از برنامه روزهای آینده را ارائه دهد.

## اجرا

```bash
python main.py
```

بعد از اجرای دستور بالا، رابط گرافیکی باز می‌شود و می‌توانید از طریق تب‌های «Tasks» و «Events» اطلاعات را مدیریت کنید.

## امکانات

- مدیریت کامل کارها و رویدادها در یک رابط گرافیکی ساده و مدرن
- اضافه کردن کارها با تعیین تاریخ سررسید، اولویت، برچسب و توضیحات اختیاری
- علامت‌گذاری کارها به‌عنوان انجام‌شده تنها با چند کلیک
- اضافه کردن رویدادها با تاریخ و زمان شروع/پایان، مکان و برچسب‌های اختیاری
- مشاهده فهرست کامل کارها (شامل انجام‌شده‌ها) و رویدادها با جزئیات
- نمایش آمار لحظه‌ای از تعداد کل کارها، انجام‌شده‌ها، سررسیدهای پیش‌رو و تعداد رویدادها

## نکات تکمیلی

- تاریخ و زمان‌ها باید با فرمت `YYYY-MM-DD HH:MM` (مثلاً `2024-05-01 20:00`) وارد شوند.
- اطلاعات پلنر به‌صورت خودکار در مسیر خانگی کاربر (پوشه `.modern_planner`) ذخیره می‌شود.
- برای هر تغییر، دکمه «Refresh» امکان به‌روزرسانی سریع فهرست‌ها را فراهم می‌کند.
